
GIT commands:

1) git init  --- инициализировать Гид в директории проекта
2) git add . --- добавить все изменения на очередь (или указать необходимые файлы и папки)
3) git commit -m 'text of commit' --- закоммитить 
4) git branch --- посмотреть - сколько есть веток
5) git checkout -b branch_name --- команда перехода в указанную ветку, -b - чтоб ветку создать
6) git log --graph --- посмотреть коммиты в данной ветке 
7) git rebase master --- перемещает флаг в мастер-ветку за последним коммитом мастер-ветки (слияние)
8) git merge branch_1 --- для слияния в мастер-ветку, создает результирующий коммит слияния веток
9) git merge --no-ff name_branch_1 --- надо сделать rebase, потом перейти в master, потом выполнить merge --no-ff
10) git pull --rebase origin --- взятьб слить инфу с мастер-ветки, --rebase - означает перемещение флага на последний коммит мастер-ветки
11) git config --global --bool pull.rebase true --- прописывает действие выше автоматически при вызове команды pull
12) git status --- посмотреть информацию о состояниях наших файлов
13) .gitignore ---'.' - значит что файл скрытый, в такой файл заносятся файлы, которые Гид будет игнорить
14) git diff --- посмотреть изменения в коммитах
14) git remote add origin https://xxxxxxxx.yyy 
15) git push -u origin master --- push-дать, впихнуть; origin - ссылка на удаленный репозиторий; master -ветка
16) git clone https://adress_of_github- загрузить с Гитхаба

Linux terminal's commands: 

1) cd .. --- перейти в директорию ниже (или указать адрес)
2)mkdir folder_name --- создать папку
3)touch file_name.xxx или > file_name.xxx --- создать файл
4) cd .. && touch text.txt --- && позволяет в одну строку записывать несколько команд
5) ls -а -l --- посмотрнть содержание директории, -а - посмотреть с скрытыми файлами, -l - росширенный просмотр
6) rm file_name.xxx --- удалить файл
7) rmdir folder_name --- удалить папку
8) apt (apt-get, pkg) install programm_name --- установка пакетов
9) vim file_name --- вызов редактора файла, выход --- :wq
10) pwd --- показывает путь, где мы находимся
11) echo message --- выводит сообщение message
12) echo message > file_name.xxx --- запись сообщения в файл
13) cat file_name.xxx --- посмотреть содержимое файла в терминале
14) Структура файловой системы Linux:
	* Папка bin - содержит исполняемые файлы всех команд Linux
	* Папка boot - содержит ядро системы, загрузчик grab
	* Папка dev - содержит файлы отвечающие за работу устройств
	* Папка home - содержит папки зарегистрированых пользователей
	* Папка lib - содержит библиотеки
	* Папка media - содержит папки с медиа
	* Папка proc - содержит папки, отвечающие за работу ядра
	* Папка root - домашняя папка суперпользователя, хранит все его настройки
	* Папка user/share/doc - по этому пути находится вся документация на все программы
15) file file_name.xxx --- посмотреть информацию о файле 
16) cp file_name.xxx file_name_copy.xxx --- копирует первый файл во второй
17) find file_name.xxx folder_name --- поиск файла в директории
18) sort  --- сортировка файлов (по умолчанию - по алфавиту названий файлов)
19) clear --- очистка экрана терминала
20) ls -l > file_name.xxx --- запись результата команды в файл, вместо ls может быть другая команда
21) echo 'date; pwd; ls' >> file_name.txt --- допишет результат команды echo в файл, в котором уже есть информация
22) sh file_name.xxx --- выполнит команды, записаные в файле (см.пункт 21)
23) date --- выводит текущую дату
24) ps --- список процессов выполняемых в даный момент - посмотреть PID_of_process
25) Cntr-Z - выход из какой то программы
    Cntr-H - посмотреть скрытык папки
    Cntr+Shift+C - копировать в термитнале
    Cntr+Shift+V - тавить в терминале
26) kill -9 PID_of_process --- вырубить процес по указанному PID
27) exit --- закрыть терминал
28) mv file_name.xxx FILE_NAME.xxx --- переименовать файл
29) allias - для создания своих комманд
	~: allias--> vi .bashrc --> alias mysite='mkdir /tmp/mysite && sshfs.......' 
	Сохранимся и надо перезапустить систему. 
30) whereis kivy-examples --- укажет куда установился пакет

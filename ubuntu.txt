
					My Ubuntu 19.10:

1) Отрезаем кусок диска и делаем неразмеченную область, например в Винде - в свойствах диска
2) Делаем загрузочную флешку - заливаем образ системы на флешку программой Rufus.
3) Входим в биос моего Dell Latitude кнопкой F2
4) Устанавливаем систему по этапам. Там где надо выбрать куда устанавливаем - выбираем Установку в 	Ручную - делаем диск загрузки 25МБ, /boot, ext2, primary; диск с системой - /, ext4, от 40ГБ, 	logical; и диск для данных - 100ГБ, ext4, /home, logical
5) Устанавливаем программу для установки deb-пакетов программ - GDebi 
(https://linuxrussia.com/gdebi-deb-ubuntu.html):
	~$  sudo apt install gdebi
6) Скачиваем с сайта Google deb-пакет гугл-хрома, нажимаем его и в окне в пункте Установить другими программами - выбрать установку через GDebi - установить Google Crome браузером по умолчанию
7) Устанавливаем TimeShift - программа для возможности отката системы (http://compizomania.blogspot.com/2017/10/timeshift-linux.html):
	~$	sudo apt-add-repository -y ppa:teejee2008/ppa
		sudo apt update
		sudo apt install timeshift
8) Делаем первую точку отката практически чистой системы для будущего через Timeshift
(https://www.youtube.com/watch?v=GlwHLnVRSBQ&list=WL&index=2&t=0s)
-----------------------------------------------------------------------------------------------

9) Устанавливаем VirtualBox и пакет росширений к ней (https://losst.ru/ustanovka-virtualbox-v-ubuntu-18-04) - скачиваем пакет для Linux и устанавливаем через меню программой GDebi, пакет расширений тоже скачиваем активируем его.
10) Устанавливаем Git (https://www.youtube.com/watch?v=f6b5p5ss1nA&list=WL&index=3):
	sudo add-apt-repository ppa:git-core/ppa
	sudo apt-get update
	sudo apt-get install git

	Для проверки версии Git выполните команду
	git --version
11) Решаем проблему кодировки символов в текстовом редакторе Gedit 
(https://www.youtube.com/watch?v=GAsyvIJiFZ8&list=WL&index=1):
	--- sudo apt install dconf-editor --- устанавливаем dconf-editor
	--- в поиске вводим dconf и вызываем эту прогу
--- заходим в: org/gnome/gedit/preferences/encodings/candidate-encodings
	--- вводим: ['UTF-8', 'WINDOWS-1251', 'CURRENT', 'ISO-8859-15', 'UTF-16']
	--- выходим из редактора - готово
Или может сработать эта строка, введенная через терминал: 'WINDOWS-1251', 'CURRENT', 'ISO-8859-15', 'UTF-16']"
12) Установили Python3, pip3, cython, kivy, buildozer, kivy-examples и необходимые зависимости
	Все прошло успешно.
13) Делаем точку восстановления №2
-----------------------------------------------------------------------------------------------
14) Устанавливаем Pycharm:
	--- скачиваем архив с сайта
	--- распаковываем его: tar -wvzf name_of_archive
	--- или правой кнопкой мыши - Extract Here...
	--- в терминале переходим по адресу: :~$ /Distributives/pycharm-community-2019.3.2/bin 
	--- запускаем установочный файл: ./pycharm.sh --- произойдет установка
	--- далее жмем next... в всплывающих окнах настроек программы
	--- устанавливаем иконку в панель: tools - create desctop..., закрыть программу и через поиск перетянуть иконку на панель.
15) Установил CentOs7_minimal на виртуальную машину
16) Удаление программ: sudo apt remove name_programm | sudo apt purge name_programm
17) xkill --- принудительное завершение программ через терминал
18) sudo reboot - принудительная перезагрузка системы через терминал
19) Cntr-H - посмотреть/скрыть скрытые папки и файлы
20) lsusb - чтоб посмотреть какие usb-устройства есть
21) Прописываем свой аппарат в библиотеках:
	- lsusb - смотрим параметры подключенного смартфона - интересует id
	- заходим в /lib/udev/rules.d - копируем оттуда файл 69-libmtp.rules
	- записываем в него строку своего смартфона: idVendor первые 4 цыфры из id (lsusb) и 			idProduct - вторые 4 цыфры
	- копируем его через sudo cp в /etc/udev/rules.d
	- /etc/init.d/udev restart --- перезапускаем etc
22) Установка Genymotion:
	- скачать дистрибутив по адресу: https://www.genymotion.com/download/
	- перейти в папку с дистрибутивом: root@linuxhelp1:~# cd /home/user1/Downloads/
	- применить эту команду: 
		root@linuxhelp1:/home/user1/Downloads# chmod +x genymotion-2.12.1-linux_x64.bin
	- установить Genymotion: 
		root@linuxhelp1:/home/user1/Downloads# ./genymotion-2.12.1-linux_x64.bin 
	- скачать архив ARM отсюда:
		https://github.com/m9rco/Genymotion_ARM_Translation/blob/master/package/
			Genymotion-ARM-Translation_for_6.0.zip
	- перетащить этот архив прям на окно эмулятора.
	- apk-файлы тестируемых приложений тоже просто перетягиваем в окно экрана эмулятора

23) Создание третьей точки отката системы - точка№3

24) pydoc modules ---посмотреть какие модули установлены в питон
25) pip freeze --- посмотреть какие модули установлены через pip
26) sudo apt install xz-utils
    tar -xf file.tar.xz --- извлечь из архива
    -x означает извлечь архивный файл
    -f означает следующее имя архивного файла
27) df - посмотреть размеры дисовогопространства
    df -h --- более читабельный вариант
    df -h /home --- посмотреть выбранный каталог
    -m --- отображать размер в мегабайтах
    -T --- отображать тип файловой системы
    -a --- включить в список все файловые системы
    -i --- отображать используемые иноды
28) du --- посмотреть сколько места занимают файлы
    du -h /home --- выводит размер файлов в выбранной директории
    -b --- выдавать размер в байтах
    -k --- показывать размер в килобайтах
    -m --- размер в мегабайтах
    -x --- не учитывать каталоги, в других файловых   системах.
